# ⚡ Smart Developer Team Management System

A full-stack team collaboration and project management platform.

## Features
- User Authentication (Register, Login, Logout, Profile)
- Skill Tracking System (Add, Level, Verify)
- Team Builder (Create, Add/Remove Members, Roles)
- Project Management (Create, Deadline, Status)
- Task Management (Kanban Board, Assign, Priority)
- Performance Analytics (Charts)
- Leaderboard
- Notifications
- Team Chat
- File Upload/Download
- Search (Users, Projects, Teams, Tasks)
- Admin Panel

## Tech Stack
- Backend: Python + Flask
- Database: SQLite
- Frontend: HTML + CSS + JavaScript + Chart.js

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/smart-dev-system.git
cd smart-dev-system
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Mac/Linux
pip install -r requirements.txt
python app.py
```

Open: http://127.0.0.1:5000
